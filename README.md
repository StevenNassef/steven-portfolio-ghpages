# Steven Nassef Henry — Portfolio

A minimal, fast React + Tailwind site ready for GitHub Pages or Vercel.

## Local Dev
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to GitHub Pages (automatic via Actions)
1. Create a new repo, push this project.
2. In GitHub: **Settings → Pages** — set **Source: GitHub Actions**.
3. The included workflow will build and publish to the `gh-pages` branch on every push to `main`.
